시간 축 시각화 (막대 그래프 / 타임라인)
"src\docs\구상예시화면.jpg" 한 시점의 스냅샷

발화가 시간에 따라 어떻게 변하는지 보고 싶으면, 각 뉴런별 spike raster plot(시간 vs 발화 여부)이나 발화율 히스토그램을 ImGui로 켜고 끌 수 있게
예) 뉴런 원을 클릭하면, 해당 뉴런의 시간 축 그래프 팝업.

시냅스 가중치 시각화
연결선 두께나 색을 가중치(weight) 크기에 매핑하면 더 직관적일 것 같음.
강한 시냅스 → 두껍게 / 밝게
약한 시냅스 → 가늘게 / 희미하게
학습이 진행되면서 선의 굵기 변화가 실시간으로 보이면 학습 과정 디버깅이 쉬워짐.

뉴런 속성 강조
사진에서는 “발화 vs 미발화”만 색상 차이가 있는데, 경쟁 수치나 활성도에 따라 노드 크기/투명도/테두리로 전환이 가능하도록 기능.
발화 → 노란색, 크기 커짐 (파열처럼 보이게)
미발화 → 회색, 작게 유지
경쟁 수치 높음 → 붉은 테두리

계층별 요약 정보
각 계층(1계층, 2계층…) 옆에 숫자 요약 패널:
발화율 평균
경쟁 성공률
입력 신호 커버리지
그래프만 보다가도 “현재 계층이 잘 동작하는지” 한눈에 확인 가능.

디버깅 툴 패널 확장
사진에서 imgui 박스에 있는 옵션은:
셰이더 전환
학습 막대 그래프 ON/OFF
여기에 추가하면 좋은 것:
뉴런 인덱스 검색 (특정 뉴런 jump)
연결 보기 필터링 (예: 특정 뉴런과 연결된 시냅스만 강조)
선 위, 또는 중간에 있는 가중치 및 경쟁수치, 그리고 데이터 입력 텍스트를 각각 ON / OFF
계층별 시뮬레이션 속도 조절 (time scale) <- 파이썬에서 주는 공유데이터의 dt값을 수정해야 할 것임

상호작용성
뉴런 원을 클릭 → 해당 뉴런 파라미터(막전위, 가중치 목록, 입력 신호 값 등) 팝업
시냅스 선 위에 hover → 연결 강도/최근 학습 변화 표시
Ctrl+드래그 → 특정 영역 줌인
