## 축삭 및 전도
1. 축삭 분기 라우팅 및 전도 실패 (Axon Branch Routing & Failure)
	- GPU 구현(AxonManager)에서는 단순 delay 큐만 있음 → SF 기반 실패 확률 반영 안 됨.
2. 단순화된 축삭 전도 지연
	- 고정 delay, 분포적 지연·빈도 의존성 없음.
3. 역전파 행동전위(BAP) 부재
	- soma→dendrite로의 backpropagation 스파이크 전도 특수 처리 없음.

## 시냅스
4. 단순화된 시냅스 모델
	- deterministic 전달, vesicle release 확률·전달 실패 없음.
	- NMDA Mg-block 단순화.
	- GPU 쪽은 AMPA/NMDA 구분조차 없음.
5. 학습 및 가소성 규칙 부재
	- STDP, Ca-LTP, 장기/단기 가소성 모두 없음.
	- 구조적 가소성(Structural Plasticity) 미구현.
	- GPU는 Python에서 있는 plasticity 기능조차 없음.
6. 시냅스 통합 다이나믹스 단순화
	- 단일 지수적 시냅스, conductance만 더해짐.
	- STP(단기 facilitation/depression) 없음.

## 뉴런 모델
7. 뉴런 모델의 세분화 부족
	- Pyramidal만 모델링, 다양한 세포 유형(예: PV, SST, VIP 인터뉴런) 없음.
8. 구획별 채널 다양성 부족
	- Na/K/Leak만 있음. Ca, HCN, 다양한 K채널, Cl- 등 부재.
	- 능동 수상돌기 전류 없음 → dendritic spike 불가.
9. 비선형적 수상돌기 연산 부재
	- local regenerative spike, NMDA plateau, branch-specific 연산 없음.
10. 스파이크 후 회복/불응기 단순화
	- Python 쪽은 간단한 refractory만 있음.
	- GPU 쪽은 미구현(스파이크 플래그만 리셋).
11. 잡음(noise)·채널 확률성 부재
	- gating 변수는 deterministic 평균식 → 실제 변동성 미반영.
12. 세포 크기·표면적 고려 단순화
	- area 값은 있지만 cable equation에 길이/직경 반영 부족.

## 네트워크 및 시스템 수준
13. 네트워크 토폴로지의 단순성
	- 단일 뉴런 또는 소규모 연결.
	- 복잡한 층 구조, 반복 회로, 모듈형 토폴로지 없음.
14. 멀티뉴런 네트워크 부재 (GPU)
	- GPU 버전은 단일 뉴런에 하드코딩.
15. 네트워크 이벤트 스케줄러 단순화
	- Python EventQueue는 있으나 GPU는 단순 host-side 큐.
	- 대규모 병렬 이벤트 처리 미비.
16. 외부 환경과의 상호작용 부재
	- Poisson generator 수준의 외부 입력만 있음.
	- 닫힌 루프 시뮬레이션 없음.
17. 전기적 연결(Gap junction) 부재
	- axial coupling은 있으나 inter-cell gap junction 없음.

## 기타 생리학적 요소
18. 신경조절(Neuromodulation) 메커니즘 부재
	- dopamine, ACh, metabotropic receptor 등 없음.
19. 온도 의존성(Q10 scaling) 없음
	- 상수는 정의돼 있지만 gating kinetics에 적용 안 됨.
20. 수치적 통합법 단순화
	- Forward Euler 전용, stiff 계 방정식에 비효율적.
